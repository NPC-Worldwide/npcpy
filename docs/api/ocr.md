# OCR

Wrapper utilities for running DeepSeek OCR via Unsloth (lazy-loaded).

::: npcpy.gen.ocr
    options:
      show_source: true
      members: true
      filters:
        - "!^_"          
        - "!^[A-Z]{2,}"  
        - "!test_"       
      inherited_members: false
      show_root_heading: false
      show_if_no_docstring: true
